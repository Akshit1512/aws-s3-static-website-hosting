# AWS S3 Static Website Hosting

## Project Overview

Hosted a static website using Amazon S3 Static Website Hosting.

## AWS Services Used

* Amazon S3
* Static Website Hosting
* Bucket Policy

## Implementation Steps

1. Created an S3 bucket
2. Uploaded website files
3. Configured bucket policy
4. Disabled Block Public Access
5. Enabled Static Website Hosting

## Challenges Faced

### AccessDenied Error

* Fixed by updating bucket permissions and public access settings.

### 404 NoSuchKey Error

* Fixed by correcting the index document name.

* Amazon S3
* Bucket Policies
* Static Website Hosting
* AWS Troubleshooting
